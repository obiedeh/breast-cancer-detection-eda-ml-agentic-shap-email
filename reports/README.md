# Reports

Generated workflow reports are written here by default under `reports/generated/`.

The checked-in sample reports show the expected markdown structure and safety language. The generated folder contains reproducible run outputs:

- `model_report.md`
- `explainability_report.md`
- `model_card.md`
- `metrics.json`
- `confusion_matrix.svg`
- `roc_curve.svg`
- `feature_importance.svg`

These are demonstration artifacts only and do not provide medical diagnosis, treatment instructions, or autonomous clinical recommendations.
